The following changes have been done in version 1.6 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULENag- (CL-um-)
finds error in:Nagbunot ng ngipin ang dentista.
NEW RULESound and Letter Change from 'd' to 'r'
finds error in:Baka ang tingin mo ay talo ka din.
finds error in:Kinuha ko ito dito.
finds error in:Natalo ka daw sa pustahan.
finds error in:Iniwan ko diyan ang pantasa.
finds error in:Baka ang tingin mo doon ay bahay ko.
NEW RULESound and Letter Change from 'r' to 'd'
finds error in:Baka ang tingin mo ay pangit rin ikaw.
finds error in:Kinuha ko itong lapis rito.
finds error in:Matapang raw ikaw.
finds error in:Iniwan ko ang lapis riyan.
finds error in:Baka ang tingin mo ay roon ang papunta sa bahay nila.
