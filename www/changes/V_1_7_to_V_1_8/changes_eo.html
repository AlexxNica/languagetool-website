The following changes have been done in version 1.8 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEŝati al → plaĉi al
finds error in:Espereble ne malŝatos al vi.
finds error in:Al mi ŝatas sportumi.
finds error in:Al mi ne ŝatas sportumi.
finds error in:Al mi ĉiam ŝatas sportumi.
finds error in:Al mi ĉiam ne malŝatas sportumi.
IMPROVED RULEestonteco
does not find error in:Mi intencas morgaŭ foriri.
IMPROVED RULEmankanta akuzativo
does not find error in:Kion vi konsideras vera?
does not find error in:Li loĝas tie sola.
does not find error in:Mi trovas tion inteligenta.
IMPROVED RULEnekongruaj nombroj
does not find error in:Ili estas granda parto de la sukceso.
does not find error in:Ni ĉiuj estas granda parto de la sukceso.
does not find error in:Piero kaj Marko estis ĉi tie.
IMPROVED RULEverbo kiu preskaŭ ĉiam postulas akuzativon
finds error in:Oni devas uzi aŭ Vindozo, aŭ Linukso.
does not find error in:Oni devas uzi aŭ Vindozon, aŭ Linukson.
REMOVED RULEjam + futuro
