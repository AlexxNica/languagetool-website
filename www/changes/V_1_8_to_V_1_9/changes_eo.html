The following changes have been done in version 1.9 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEdefekto → difekto
finds error in:Mi havas neniu defektojn.
NEW RULEdum kiam → dum
finds error in:Tio okazas dum kiam mi estas hejme.
finds error in:Tio okazas dum tio ke mi estas hejme.
finds error in:Tio okazas dum tio, ke mi estas hejme.
NEW RULEeksterne → ekstere
finds error in:Li ludas eksterne.
NEW RULEenloĝanto → loĝanto
finds error in:La enloĝantoj.
NEW RULEkoncerne + akuzativo
finds error in:Inkluzive ili
finds error in:Escepte tiu homo
finds error in:Koncerne la riskoj
finds error in:Inkluzive la studentoj
finds error in:Escepte tri studentoj
NEW RULEmagika → magia
finds error in:Estis magika vespero.
NEW RULEnavalida vorto unun
finds error in:Mi aĉetis unun panon.
NEW RULEneĝusta ĉapelo.
finds error in:Ankaû mi venos.
finds error in:Ankaǔ mi venos.
NEW RULEplasta kaj plastika
finds error in:Ĉu vi havas plastikan sakon?
NEW RULEpro tio, ĉar → pro tio, ke
finds error in:La infano ploras pro tio, ĉar li volas manĝi.
finds error in:La infano ploras pro tio ĉar li volas manĝi.
NEW RULEversiono → versio
finds error in:La nova versiono tre bonas.
NEW RULEx-sistemo → Unikodo
finds error in:Cxi tiu frazo ne uzas Unikodon.
finds error in:CXi tiu frazo ne uzas Unikodon.
finds error in:Ankaux.
finds error in:Ankaux.
IMPROVED RULELitero (Letero)
does not find error in:Por ricevi la literon Ĥ, longe premu H.
IMPROVED RULEVolas ke + verbo
does not find error in:Ĉu vi volas nei ke ili ekzistas?
IMPROVED RULEe-vorto anstataŭ a-vorto
does not find error in:La legantaro konsistas el maksimume trideko da personoj.
IMPROVED RULEnetransitiva verbo + akuzativo
finds error in:Mi vivis agrablajn feriojn.
does not find error in:Mi havis agrablajn feriojn.
does not find error in:Mi vivis agrablan vivon.
IMPROVED RULEĉi-tiu (ĉi tiu)
finds error in:Mi deziras tiun-ĉi.
does not find error in:Mi deziras tiun ĉi.
REMOVED RULELa pli (la plej)
REMOVED RULETiom kiom (tiel kiom)
