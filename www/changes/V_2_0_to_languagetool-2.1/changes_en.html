The following changes have been done for English in version trunk of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEUse of continuous form with non-action verbs
finds error in:He had been knowing it.
finds error in:She was knowing it.
finds error in:She is knowing it.
finds error in:I am knowing it.
NEW RULEask wether/weather (whether)
finds error in:Don't ask me wether it is already decided!
NEW RULEdoes ... 3rd person verb
finds error in:He does not has expertise about that.
finds error in:Does anybody has an idea about that?
NEW RULEdon't ... 3rd person verb
finds error in:I don't needs help with that.
finds error in:You don't has a bicycle?
finds error in:I do not needs help with that.
finds error in:You do not has a bicycle?
NEW RULEknew (new)
finds error in:This is very knew to me.
NEW RULEprice (prize)
finds error in:She won a price.
finds error in:That is the consolation price.
finds error in:These are the consolation prices.
finds error in:That is the Nobel Price.
NEW RULEprize (price)
finds error in:This is called prize spiral.
IMPROVED RULEboth... as well as (and)
does not find error in:He is both rich and smart as well as handsome.
does not find error in:He is both very rich and smart as well as handsome.
IMPROVED RULEchance (change) a setting
finds error in:The climate chance must be stopped.
does not find error in:The climate change must be stopped.
IMPROVED RULEhe (the)
does not find error in:But there was a fire burning inside him ever since he saw those pictures on her camera.
