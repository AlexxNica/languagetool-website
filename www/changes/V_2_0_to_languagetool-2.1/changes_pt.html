The following changes have been done in version trunk of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEa gente vai
finds error in:Hoje a gente vamos à praia.
NEW RULEabrirmos
finds error in:Pediram para abrir-mos a porta.
NEW RULEdecreto-lei
finds error in:Saiu um novo decreto de lei.
NEW RULEelas têm
finds error in:Há palavras que elas tem medo de ouvir.
NEW RULEeles têm
finds error in:Há palavras que eles tem medo de ouvir.
NEW RULEentre mim
finds error in:O que se passou entre eu e o Rui?
NEW RULEmandado de busca
finds error in:A polícia tem um mandato de busca.
NEW RULEnão vês que
finds error in:Pai, não vez que eles estão errados?
NEW RULEà sua
finds error in:Ele deu uma mãozinha há sua família.
NEW RULEà tua
finds error in:Dá uma mãozinha há tua família.
