The following changes have been done in the latest version of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEadulto kaj adolto
finds error in:Li estas jam adulto.
finds error in:Adultoj kaj liaj infanoj rajtas partopreni.
finds error in:Infanoj kaj adultoj partoprenis.
finds error in:Maria baldaŭ fariĝos adultino.
NEW RULEnavalida pluralo ĉe e-vorto.
finds error in:Mi kuras vesperej.
NEW RULEregardi kaj rigardi
finds error in:Ŝi regardis lin.
NEW RULEsi subjekto
finds error in:Sia frato alvenis.
finds error in:Siaj fratinoj alvenis.
finds error in:Tom kaj sia frato alvenis.
finds error in:Tom kaj sia fratino alvenis.
IMPROVED RULEIo + O-vorto
finds error in:Temas pri io libro.
does not find error in:Temas pri iu libro.
IMPROVED RULEPrepozicio je la fino de frazo
does not find error in:Ĝis!
