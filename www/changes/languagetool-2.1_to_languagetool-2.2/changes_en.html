The following changes have been done for English in version 2.2 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEI was wandering (wondering) if
finds error in:I was wandering if someone could help me.
NEW RULEI'm note (not) sure
finds error in:I am note sure.
finds error in:I'm note sure.
NEW RULEUse of past form with 'going to ...'
finds error in:I'm going to wrote him.
NEW RULEif it if (is)
finds error in:I wonder if the context if actually important.
finds error in:The rules are testes, which if quite fast.
NEW RULEmake us (use) of
finds error in:He made us of the new tools.
NEW RULEon (at) first glance
finds error in:On first glance it seems to be a good idea.
NEW RULEon (one) of the most/least/...
finds error in:It's on of the most popular songs.
IMPROVED RULE'many/few' + uncountable noun, e.g. 'many (much) food'
finds error in:She has many money.
does not find error in:She has much money.
does not find error in:There are many food plants.
IMPROVED RULESmart quotes (“”)
finds error in:``Test”.
finds error in:„Test”.
does not find error in:“Test”.
does not find error in:“Test”.
IMPROVED RULEfro a while (for a while)
finds error in:That's good fro your.
finds error in:And good fro your.
finds error in:Fro me.
does not find error in:For me.
does not find error in:And for me.
does not find error in:That's good for your.
does not find error in:For me.
