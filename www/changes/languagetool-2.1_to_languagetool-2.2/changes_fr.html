The following changes have been done in version 2.2 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEde puis → depuis
finds error in:J’habite de puis longtemps à Paris.
NEW RULEeux-même → eux-mêmes
finds error in:Elles y arrivent elles-même.
NEW RULEil nous a informé → il nous a informés
finds error in:Il nous a informé.
finds error in:Il nous a informée.
finds error in:Dominique les a renseigné.
finds error in:Dominique les a renseignée.
NEW RULEla → là
finds error in:Il l’a trouvé la où je l’avais laissé.
NEW RULEvénéneux ou venimeux
finds error in:Des araignées vénéneuses.
finds error in:Les couleuvres ne sont pas vénéneuses.
finds error in:Des champignons non venimeux.
finds error in:Ces plantes ne sont pas venimeuses.
IMPROVED RULEAccord en nombre
does not find error in:1 fois.
does not find error in:Il a battu le record avec un temps de 3 min 18 s.
does not find error in:Ces don Juan.
IMPROVED RULEComment + verb ppa
does not find error in:Je me suis en fait amusé.
IMPROVED RULETrait d’union dans l’inversion verbe-pronom sujet
does not find error in:L’histoire commence il y a longtemps.
IMPROVED RULEVerbe conjugué après pronom relatif qui
finds error in:Et moi qui pensait qu’il m’aimait.
does not find error in:Et moi qui pensais qu’il m’aimait.
does not find error in:Il n’y avait qu’elle et moi qui savions.
IMPROVED RULEVerbe conjugué suivi d’un verbe conjugué
does not find error in:L’enquête ne fait que commencer.
IMPROVED RULEaccord que + verbe avoir
does not find error in:L’expert estime que les climatosceptiques ont fait pression.
IMPROVED RULEen-bas → en bas (etc.)
finds error in:Les trois-quarts du temps, il ne fait rien.
finds error in:La bouteille est aux trois-quarts pleine.
does not find error in:Les trois quarts du temps, il ne fait rien.
does not find error in:Un trois-quarts de rugby.
does not find error in:Les trois-quarts de l’équipe de rugby.
does not find error in:La bouteille est aux trois quarts pleine.
does not find error in:Un trois-quarts de rugby.
IMPROVED RULEvoie → voix
does not find error in:Cette décision ouvre la voie à des élections.
IMPROVED RULEêtre + participe présent
does not find error in:Ils sont bien sûr ressortis indemnes.
