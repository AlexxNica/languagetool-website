The following changes have been done for English in version trunk of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULE'allow' + 'to' + infinitive
finds error in:It allows to see better.
finds error in:It allowed to see better.
NEW RULE'did' with past tense verb
finds error in:Did the application worked?
finds error in:Did the applications worked?
finds error in:Did your new applications worked?
NEW RULEWord order: Wrong position of adverb, e.g. 'Always I am happy. (I am always happy.)'
finds error in:Always I am happy.
finds error in:Ever it was thus.
finds error in:I am happy always.
finds error in:I sometimes am happy.
finds error in:I am here and I sometimes am happy.
finds error in:They have been seen often together.
finds error in:They often have been seen together.
finds error in:They have been often seen together.
finds error in:I go often to bed before midnight.
finds error in:I will love always you.
finds error in:I always will love you.
NEW RULEWordiness: in a ... manner
finds error in:He came in a hasty manner.
NEW RULEYOUR_SHOULD
finds error in:If it's running, your should deactivate it.
NEW RULEa through(thorough) discussion etc.
finds error in:The manual does not have a through discussion of the topic.
NEW RULEa was (way)
finds error in:But is there a was to work around the problem?
NEW RULEas to whether or not (whether or not)
finds error in:As to whether or not he lives...
NEW RULEconfusion of our/out
finds error in:We evaluated out method by running RASP.
NEW RULEthe some (same) day
finds error in:It happened on the some day.
NEW RULEwildly (widely) accepted
finds error in:This idea is now wildly accepted.
IMPROVED RULEPossible agreement error 'there're' + singular noun
finds error in:There're problem in Iraq but it is not a terrible situation.
finds error in:There're a lot problem in Iraq but it is not a terrible situation.
finds error in:There're many problem in Iraq but it is not a terrible situation.
finds error in:There are many problem in Iraq but it is not a terrible situation.
finds error in:There are a lot problem in Iraq but it is not a terrible situation.
does not find error in:There're many problems in Iraq.
does not find error in:Other than this, there are no known surviving texts.
does not find error in:There are many other deliberately insulting terms.
does not find error in:There are a variety of other types.
does not find error in:There are ten known isoforms of adenylate cyclases in mammals.
IMPROVED RULEPossible agreement error: numeral + singular countable noun
finds error in:I have five book.
finds error in:Additionally, there are over 500 college and university chapter.
finds error in:There are 500 college and university chapter.
does not find error in:US$10,000 for a single 25 minute episode.
does not find error in:Thomas Watson at 333 Grant Avenue in San Francisco.
does not find error in:The 16 year old Ahmad Shah Durrani.
does not find error in:A dramatic series in 1956 saw a record that will probably never be beaten.
IMPROVED RULEUse of past form with 'going to ...'
does not find error in:We are not going to let them get away with this.
IMPROVED RULEits (it's)
finds error in:It does this by providing it's own open source based products.
does not find error in:It does this by providing its own open source based products.
IMPROVED RULEless/more ... then (than)
finds error in:It's a less controversial decision then one would think.
does not find error in:It's a less controversial decision than one would think.
IMPROVED RULEto/two (too) late/soon/much
does not find error in:It's limited to much shorter terms.
