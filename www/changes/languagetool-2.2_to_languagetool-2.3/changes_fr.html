The following changes have been done in version 2.3 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEen particuliers -> en particulier
finds error in:Le métaux précieux et l’or en particuliers restent des investissements sûrs.
NEW RULEqu il → qu’il
finds error in:Je pense qu il ment.
NEW RULEsceau et seau
finds error in:Un sceau d’eau.
finds error in:Ton sceau est percé.
finds error in:Le seau du secret.
NEW RULEva-t-en → va-t’en
finds error in:Va-t-en !
NEW RULEça et là → çà et là
finds error in:Il trouva de la nourriture ça et là.
IMPROVED RULEAccord en genre
does not find error in:J’ai grand faim.
IMPROVED RULEPeut ou peux au lieu de peu
does not find error in:Je veux m'éloigner autant que faire se peut d'ici.
IMPROVED RULEVerbe conjugué après pronom relatif qui
does not find error in:J’ai un fils qui est en secondaire.
does not find error in:Quelques jeux vidéo qui ont marqué les années 80.
IMPROVED RULEVerbe conjugué suivi d’un verbe conjugué
does not find error in:La ville, cité interdite.
does not find error in:Puis soutenue par des bombardement, il attaqua.
does not find error in:Elles semblent décrites par le modèle en couches.
does not find error in:L’influence acquise.
does not find error in:Quelqu’un a pris ma place.
IMPROVED RULEaccord que + verbe avoir
does not find error in:De la même nature que celles qui ont été saisies.
IMPROVED RULElà sans trait d’union
does not find error in:Tout est là.
IMPROVED RULEpronom sujet + nom/adj
does not find error in:On ne la paie pas.
IMPROVED RULEsujet singulier + verbe pluriel
does not find error in:Des petites bites.
IMPROVED RULEverbe après préposition
does not find error in:La pile de Volta.
does not find error in:Voici venu le temps des vacances.
