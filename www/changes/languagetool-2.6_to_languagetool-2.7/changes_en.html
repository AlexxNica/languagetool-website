The following changes have been done for English in version 2.7 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULE'The' or 'a' before a punctuation
finds error in:I am going to an, then I see you.
finds error in:I am going to the; that's right.
NEW RULEMay 20001 (May 2001)
finds error in:I will go to Holland in May 20014.
NEW RULEMissing definite article in the date
finds error in:It happened in 1990s.
finds error in:It happened in 1990's.
NEW RULEMissing definite article referring to a time period
finds error in:Something happened in past.
NEW RULEMissing space after closing quote
finds error in:“Good morning, Frank,”said Hal.
NEW RULEReadability: Three nouns in a row
finds error in:The security protection software we offer is the state-of-the art achievement.
NEW RULESentence is a fragment
finds error in:Though the dog likes to bark.
finds error in:After we had dinner.
finds error in:Even if the dog likes to bark.
finds error in:So if the dog likes to bark.
finds error in:So, if the dog likes to bark.
finds error in:So even if the dog likes to bark.
finds error in:So, even if the dog likes to bark.
finds error in:In order that the dog guards the house.
finds error in:Provided that the dog guards my house.
NEW RULESentence starting with 'And' or 'But'
finds error in:And yes, it shows a lack of style to start a sentence with 'and'.
finds error in:But it is not a great writing style to start a sentence with 'but'.
