The following changes have been done in version 2.7 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULE*surgir/surtir efecto
finds error in:No surgió efecto.
NEW RULEEl agravante y el atenuante
finds error in:Se hizo con el agravante de nocturnidad.
finds error in:En este caso concurre el atenuante del pronto pago.
NEW RULEté sustantivo
finds error in:El te verde japonés es muy sabroso.
IMPROVED RULEa + participio
does not find error in:Este bacalao huele a pescado fresco.
REMOVED RULEté_sustantivo
