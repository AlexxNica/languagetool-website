The following changes have been done in version 2.8 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULE*ha/a + infinitivo
finds error in:Lo ha vuelto ha hacer.
finds error in:Volver ha casa.
NEW RULEEl + conjunción
finds error in:Esto queda entre el y yo.
NEW RULESe + hacia es se hacía
finds error in:Eso se hacia mal.
finds error in:Eso le hacia mal.
finds error in:Esa bota la hacia otra compañía.
NEW RULESe delante de ciertas palabas
finds error in:Qué se yo.
finds error in:Pero se quién fue.
NEW RULETu + conjunción
finds error in:Esto queda entre tu y yo.
NEW RULEUso incorrecto del plural del verbo haber
finds error in:Hubieron varios disparos
