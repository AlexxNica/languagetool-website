The following changes have been done in version 2.8 of LanguageTool (xml-based rules only):
No new rules have been added and no rules have been removed.
NEW RULEnombre mille invariable
finds error in:J’ai gagné dix milles euros.
finds error in:Trois-milles euros.
finds error in:trois-milles-neuf-cent-douze
finds error in:milles-deux-cents euros.
IMPROVED RULEAccord en genre
does not find error in:J’habite 10 avenue Constantin Arnaud.
IMPROVED RULEDes, dès, dés
does not find error in:Plusieurs des caractéristiques.
IMPROVED RULEJimmy Hendrix → Jimi Hendrix (etc.)
finds error in:André Agassi.
IMPROVED RULEmois de l’année
does not find error in:La monarchie de Juillet.
IMPROVED RULEpas de verbe après : au, de, du…
does not find error in:Ils ont des a priori sur nous.
IMPROVED RULEvoire même
does not find error in:On ne veut pas voir la même chose.
REMOVED RULEarc en ciel
REMOVED RULElui même
